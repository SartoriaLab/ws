# Imagens — Wilson's Pizzaria

## Já no repositório
- `logo.png` — extraído do EatFood (https://ajx.nyc3.digitaloceanspaces.com/appsconfig/logos/WILSONSPIZZARIATAQUARITINGA7.webp)

## Pendente do cliente

Para substituir os placeholders Unsplash usados em `index.html`/`styles/main.css`:

1. **Hero** (`hero.jpg` ou `.webp`) — foto fullbleed de uma pizza recheada (1600px largura mínima). Usado em `.hero-bg` no CSS. Após enviar, trocar URL em `styles/main.css` linha do `.hero-bg`.
2. **Sobre** (`ambiente.jpg` ou `.webp`) — foto da fachada, salão ou mesas externas (900px+). Trocar em `.sobre-photo` no CSS.
3. **Logo alta resolução** — versão SVG ou PNG 512×512 para favicon e PWA.

## Sugestão de formatos
- `.webp` para fotos (qualidade 80, melhor compressão)
- `.svg` ou `.png` para logo
- Renomear sem espaços, tudo minúsculo: `hero.webp`, `ambiente.webp`, `logo-512.png`
